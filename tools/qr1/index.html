Better Qr Codes

Spring, 2024
web tool

First attempt to make QR codes more presentable graphically, using a custom python pipeline. I was wondering how they can hint at what's linked in them, or at least have some personality, while still being readable with all the default scanners built into phones. Lastly, I was figuring out the actual critical elements for machine recognition and ways to push against that edge.
