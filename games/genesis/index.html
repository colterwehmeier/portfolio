Genesis

2023
chaingedsolohaunted-ps1gamecollaborationhaunted-ps1
Genesis was my submission for the C.H.A.I.N.G.E.D. collaboration ↗, drawing inspiration from vector graphics for a light gun game that reimagines the credit crawl game from Super Smash Bros Melee (2001). This project served as a platform to explore new technologies, including motion-captured camera techniques, synchronizing gameplay with music, a blocking/scripting system for sequencing events, and vector-based rendering.
