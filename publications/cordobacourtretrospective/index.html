Córdoba Court Report

Winter, 2023
game retrospectivereportanalyticsgames
Building off the People's Tree experiments, we developed a virtual courtyard space online. Within this space, players can read and submit messages attached to small decorative plates and plants. This report ↗ analyses the content posted and examines some of the behaviors observed within the space.
